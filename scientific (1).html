Scientific Calculator

Type a full expression — the navy dragon inhales every key, then breathes the answer back out.

This free online scientific calculator supports trigonometric functions (sine, cosine, tangent), logarithms, exponents, square roots, and full parentheses for order of operations. Great for algebra, trigonometry, and everyday scientific calculations — no download required.

What each key does

sin / cos / tan
Trig functions — input treated as radians.
log / ln
Base-10 log, and natural log.
^
Raise the number before it to the power that follows.
√
Square root of whatever's inside the parentheses.
π / e
